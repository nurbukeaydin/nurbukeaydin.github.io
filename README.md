# Nurbüke Aydın - Portfolio Website

A modern, responsive portfolio website showcasing my work as a Computer Engineering student with expertise in iOS development, web development, and databases.

## 🚀 Features

- **Responsive Design**: Fully optimized for all devices (desktop, tablet, mobile)
- **Smooth Animations**: Elegant transitions and scroll effects
- **Modern UI/UX**: Clean and professional design with gradient accents
- **Interactive Elements**: Hover effects and smooth navigation
- **Social Integration**: Direct links to LinkedIn, GitHub, and email
- **CV Download**: Easy access to download resume/CV
- **Scroll Progress Bar**: Visual indicator of page scroll position

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Advanced styling with gradients, animations, and responsive layouts
- **JavaScript**: Interactive features and scroll animations
- **Git & GitHub Pages**: Version control and deployment

## 📦 Installation & Setup

### Clone the Repository
```bash
git clone https://github.com/nurbukeaydin/nurbukeaydin.github.io.git
cd nurbukeaydin.github.io
```

### Local Development
```bash
# Start a local server (Python)
python3 -m http.server 8000

# Or using Node.js
npx http-server
```

Open `http://localhost:8000` in your browser.

## 🎨 Customization

### Update Personal Information
- Edit `index.html` to modify content
- Update social media links in the Contact section
- Replace `CV_NurbükeAydın.pdf` with your own resume

### Styling
- Global styles: `style.css`
- Color scheme: Purple gradient (`#9D7BB3` to `#D4A5D4`)
- Responsive breakpoints: 1024px, 768px, 480px

### Navigation
Update the navigation menu in `index.html` to add/remove sections:
```html
<li><a href="#section-id">Section Name</a></li>
```
